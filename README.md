## 安装指导

安装依赖

```
npm install

淘宝镜像
npm install --registry=https://registry.npm.taobao.org
```

启动项目

```
npm run serve
```

打包发布

```
npm run build
```

更多自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).
